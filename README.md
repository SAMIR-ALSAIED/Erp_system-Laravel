# ERP System – Laravel

ERP (Enterprise Resource Planning) system built with Laravel.

## 🚀 Features
- Authentication
- Users Management
- Categories Management
- Products Management
- Database Seeders
- Clean Laravel Structure

## 🛠 Tech Stack
- Laravel
- PHP
- MySQL
- Bootstrap
- Seeder & Factory

## ⚙️ Installation

```bash
git clone https://github.com/SAMIR-ALSAIED/Erp_system-Laravel.git
cd Erp_system-Laravel
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
